# Bookstore Management System

## Περιγραφή
Πρόγραμμα διαχείρισης βιβλιοπωλείου σε C που υποστηρίζει:
- Εισαγωγή/διαγραφή συγγραφέων και βιβλίων
- Αναζήτηση συγγραφέων και βιβλίων
- Αυτόματη ταξινόμηση δεδομένων
- Binary search για γρήγορη αναζήτηση
- Δυναμική διαχείριση μνήμης (malloc/realloc)

## Compilation

### Windows (με MinGW ή MSVC)
```bash
gcc bookstore.c -o bookstore.exe
```

### Linux/Mac
```bash
gcc bookstore.c -o bookstore
```

## Εκτέλεση
```bash
./bookstore.exe    # Windows
./bookstore        # Linux/Mac
```

## Δομή Αρχείων

### books.txt
```
<πλήθος βιβλίων>
<τίτλος βιβλίου 1>
<έτος έκδοσης>
<τιμή>
<τίτλος βιβλίου 2>
...
```

### authors.txt
```
<πλήθος συγγραφέων>
<ID συγγραφέα 1>
<επώνυμο>
<όνομα>
<αριθμός βιβλίων>
<ID συγγραφέα 2>
...
```

### writes.txt
```
<πλήθος εγγραφών>
<τίτλος βιβλίου 1>
<ID συγγραφέα>
<τίτλος βιβλίου 2>
...
```

## Λειτουργίες

1. **Insert new writer record**: Εισαγωγή νέου συγγραφέα
   - Αυτόματη ανάθεση ID
   - Ταξινόμηση βάσει ID

2. **Insert new book record**: Εισαγωγή νέου βιβλίου
   - Αυτόματη δημιουργία συγγραφέα αν δεν υπάρχει
   - Ενημέρωση πίνακα writes
   - Ταξινόμηση βάσει τίτλου

3. **Search a writer record**: Αναζήτηση συγγραφέα
   - Αναζήτηση με επώνυμο
   - Εμφάνιση όλων των βιβλίων του

4. **Search a book record**: Αναζήτηση βιβλίου
   - Binary search με τίτλο
   - Εμφάνιση συγγραφέων

5. **Delete a writer record**: Διαγραφή συγγραφέα
   - Διαγραφή μονογραφιών
   - Ενημέρωση όλων των πινάκων

6. **Delete a book record**: Διαγραφή βιβλίου
   - Ενημέρωση αριθμού βιβλίων συγγραφέων
   - Ενημέρωση πίνακα writes

7. **Exit**: Έξοδος και αποθήκευση δεδομένων

## Χαρακτηριστικά Υλοποίησης

- **Δυναμική Μνήμη**: Χρήση malloc/realloc για δυναμικούς πίνακες
- **Ταξινόμηση**: 
  - Βιβλία ταξινομημένα αλφαβητικά (τίτλος)
  - Συγγραφείς ταξινομημένοι κατά ID
  - Writes ταξινομημένα κατά writer_id και τίτλο
- **Αναζήτηση**: Binary search όπου είναι εφικτό
- **Διαχείριση Μνήμης**: Αυτόματη απελευθέρωση μνήμης κατά την έξοδο

## Παράδειγμα Χρήσης

1. Εκκίνηση προγράμματος
2. Επιλογή λειτουργίας από το μενού (1-7)
3. Εισαγωγή ζητούμενων δεδομένων
4. Επιλογή 7 για έξοδος και αποθήκευση

## Σημειώσεις

- Τα αρχεία δημιουργούνται αυτόματα αν δεν υπάρχουν
- Όλες οι αλλαγές αποθηκεύονται κατά την έξοδο
- Τα δεδομένα φορτώνονται αυτόματα κατά την εκκίνηση
